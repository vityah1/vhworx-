### 💰 [FinMan](https://finman.vhworx.com) — Personal Finance Manager

FinMan is a smart personal finance app for individuals, families, and landlords. It automates the routine so you spend less time on bookkeeping:

- **AI bank import** — upload statements from any bank; AI auto-categorizes transactions
- **Utility meter readings** via Telegram — just send a photo, AI reads the meter
- **Car expenses & mileage tracking** — fuel cost per km/mile, auto-calculated
- **Family budget** — shared access for household members
- **Multi-currency & multi-language** support

🔗 **Try it free:** [finman.vhworx.com](https://finman.vhworx.com)
